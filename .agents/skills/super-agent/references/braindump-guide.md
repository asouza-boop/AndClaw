# Guia de Braindump — Super Agente

Instruções detalhadas para a condução das 6 perguntas estratégicas.

## Objetivo
Extrair o máximo de redundância e clareza antes de gerar o arquivo do agente.

## As 6 Perguntas (Roteiro)

1. 🎯 **O Problema**
   - *Pergunta:* "Qual dor esse agente alivia? O que aconteceria se ele não existisse?"
   - *Meta:* Definir a Proposta de Valor.

2. 🧠 **A Personalidade**
   - *Pergunta:* "Qual o tom de voz? Ele é um conselheiro sábio, um executor agressivo ou um assistente invisível?"
   - *Meta:* Definir Persona e Estilo de Resposta.

3. ⚡ **As Skills**
   - *Pergunta:* "Quais são as 3 ações concretas que ele executa melhor que ninguém?"
   - *Meta:* Definir o escopo técnico.

4. 📂 **O Contexto**
   - *Pergunta:* "Quais documentos, links ou bases de dados ele precisa 'ler' para ser bom?"
   - *Meta:* Definir dependências de dados.

5. 💬 **A Conversa Ideal**
   - *Pergunta:* "Me dê um exemplo de uma pergunta do usuário e como seria a resposta de ouro dele."
   - *Meta:* Gerar o "Exemplo Prático" da Skill principal.

6. 🔗 **As Conexões**
   - *Pergunta:* "Ele deve enviar dados para o Notion? Deve consultar outro agente já existente?"
   - *Meta:* Definir interconexões do ecossistema.

## Dicas para o Super Agente
- Se o usuário parecer confuso, sugira 2 opções baseadas no nome do agente.
- Sempre valide a resposta anterior com um resumo: *"Entendi que a personalidade dele é [X]. Próxima agora..."*
